
# SocialConnect App

SocialConnect is a modern social media application built with **Kotlin and Jetpack Compose**.
The app allows users to create posts, interact with other users, manage profiles, and communicate through real-time features.

The project follows industry-standard Android development practices using **Clean Architecture, MVVM, Repository Pattern, Dependency Injection, and Firebase services.**

---

## 🚀 Features

## Phase 1 Features

### 🔐 Authentication

* User Sign Up
* User Login
* Firebase Authentication
* Secure user session handling

### 📝 Post Management

* Create new posts
* Upload post content
* View posts from users
* Like posts
* Comment system

### 👥 User Features

* User Profile
* Follow / Unfollow users
* View user posts
* User interaction system

### 🔔 Notifications

* Real-time notifications
* Like notifications
* Comment notifications
* Follow notifications

# 🏗️ Architecture

The application follows **Clean Architecture** principles with:

* MVVM Architecture
* Repository Pattern
* Separation of concerns
* Scalable and maintainable code structure

Architecture Layers:

```
Presentation Layer
        |
Domain Layer
        |
Data Layer
```
